# 📚 Wikipedia Search App

A simple and responsive **Wikipedia Search Application** built using **HTML, CSS, JavaScript, Bootstrap, and Fetch API**. Users can search any topic and instantly view Wikipedia search results.

##  How It Works

1. Type a keyword in the search box.
2. Press **Enter**.
3. The app sends a request to the Wikipedia Search API.
4. Search results are displayed with:
   - Title
   - URL
   - Description
5. Clicking a title opens the Wikipedia page in a new tab.
